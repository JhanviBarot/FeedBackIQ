## FeedbackIQ

FeedbackIQ is a sentiment-aware customer feedback intelligence platform built with Python and Streamlit. It classifies unstructured customer reviews across seven dimensions using an LLM, renders a live analytics dashboard, and generates a structured AI business action plan from the aggregated results.

**Run the app:**

```
streamlit run streamlit_ui/app.py
```

**Run the test suite (zero API cost):**

```
python test_pipeline.py
```
